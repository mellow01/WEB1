HTML 공부


HTML (Hypertext Markup Language)는 웹 페이지를 만들기 위하여 사용하는 기본적인 웹 언어의 한 종류입니다. (html 자습서 이동하기)

head태그에 쓰이는 것들
a 태그는 한 페이지에서 다른 링크를 사용하는 하이퍼 링크를 정의합니다.
a 요소 의 가장 중요한 속성은 href링크의 목적지를 나타내는 속성입니다. (주소를 입력하면 웹에서 그 주소로 링크 된다.)
target속성 지정은 어디에서 링크 된 문서를 엽니다.(_blank: 링크된 문서를 새 창이나 탭에서 엽니다.)
title 태그는 문서의 제목을 정의합니다. 제목은 텍스트 전용이어야 하며 브라우저의 제목 표시줄이나 페이지의 탭에 표시됩니다. (head 태그에 씀, html 연습장)
style속성은 색상, 글꼴, 크기 등과 같은 요소에 스타일을 추가하는 데 사용됩니다.

h1 태그는 HTML 제목을 정의하는 데 사용됩니다. h1~6 정도있고 숫자 가 커질수록 글 크기가 작아집니다.


태그 (HTML Tag List) 사이에 원하는 테그를 넣으면 됨)란 웹을 지배하는 가장 중요한 문법입니다. (위 사진 참고)
strong 태그는 내부의 내용은 일반적으로 굵게 표시합니다.
u 태그는 밑줄을 긋는 태그입니다.
br 태그는 하나의 줄 바꿈을 삽입합니다.

p 태그는 단락을 나누는 태그입니다. br 태그로 단락을 나누면 원하는 만큼의 단락을 나눌수 있지만 p 태그는 정해져있는단락의 단위가 있기 때문에 단락의 자유가 떨어집니다.
하지만 css를 사용하면 훨신더 정교하게 설정이 가능합니다.

img 태그는 HTML 페이지에 포함 된 이미지에 사용됩니다. src-이미지의 경로를 지정합니다. alt-어떤 이유로 이미지를 표시할 수 없는 경우 이미지의 대체 텍스트를 지정합니다. width-사이즈를 정할수 있습니다.


li 태그는 태그 목록 항목을 정의합니다.(위 이미지 참고)
ul 태그는 정렬되지 않은 (글 머리 기호) 목록을 정의합니다.(위 이미지 참고)
ol 태그는 정렬 된 목록을 정의합니다. 순서가 지정된 목록은 숫자 또는 알파벳순일 수 있습니다. (위 이미지 참고)
