CSS 공부


CSS는 Cascading Style Sheets의 약자입니다.
CSS는 HTML 요소가 화면, 종이 또는 기타 미디어에 표시되는 방식을 설명합니다.
CSS는 많은 작업을 절약합니다. 한 번에 여러 웹 페이지의 레이아웃을 제어할 수 있습니다.
외부 스타일시트는 CSS 파일에 저장됩니다.
CSS는 HTML 문서의 스타일을 지정하는 데 사용하는 언어입니다.
CSS는 HTML 요소가 표시되는 방식을 설명합니다.
이 튜토리얼은 CSS를 기초부터 고급까지 가르칩니다.


a는 이 페이지에 있는 모든 태그를 선택한다는 뜻에서 선택자 selector라고 불리다.
color(Property): red(Value)는 선택자(a)가 지정하는 태그들에 대해서 어떤 효과를 줄것인가에 해당되는 것이다.
선언 또는 Declatation 또는 효과라고 많이 얘기한다.
우선순위: id > class > tag selector (선택적 <---> 포괄적)
아이디 값이 active인 tag a --> 위에 쟤 하나만 active라는 아이디값을 가질 수 있음.
font-size: 60px;으로 폰트 크기를 조정할 수 있고 px, % 등 다양한 종류의 조정값이 있다.
text-align: center;으로 텍스트를 정렬할 수 있는데 기본값부터 상, 하, 좌, 우, 중앙 등 다양하게 조정이 가능하다.


CSS 테두리 속성을 사용하면 요소 테두리의 스타일, 너비 및 색상을 지정할 수 있습니다.
color 폰트 색상 을 정할 수 있게 하는거
text는 글에 관한거
decoration 데코레이션
none 끄는거
다 합해서 a로 묵여있는 폰트 컬러는 검정색으로 하면서 밑줄은 지우는 CSS
